# Rydify - Vehicle Rental System

## Overview
Rydify is a Full Stack Car & Bike Rental System developed during my Full Stack Web Development Internship. The application allows users to browse, search, and book vehicles online with a simple and user-friendly interface.

## Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL
- Tools: Git, GitHub

## Features
- User Registration & Login
- Vehicle Listing and Search
- Car & Bike Booking System
- User Profile Management
- Booking Confirmation
- Invoice Generation
- Admin Dashboard
- Responsive User Interface

## Project Structure
- Frontend Source Code
- Backend APIs
- MySQL Database Script

## Installation

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
npm install
npm start
```

### Database
Import the provided SQL file into MySQL and update database credentials in the environment configuration.
